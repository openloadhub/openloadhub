# Security Policy

## Supported Versions

OpenLoadHub v0.1 alpha receives best-effort security fixes while the public alpha is active.

## Reporting A Vulnerability

Please report security issues privately by opening a GitHub security advisory in the public repository.

Do not include secrets, private endpoints, customer data, or exploit payloads in public issues.

## Demo Credentials

The default demo accounts are for local evaluation only:

- `admin` / `ptp_demo_admin`
- `demo_tester` / `ptp_demo_tester`

Change these values before any shared deployment, and keep self registration disabled unless you have added your own account governance.
